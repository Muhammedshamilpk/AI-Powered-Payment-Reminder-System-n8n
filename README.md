**AI-Powered Payment Reminder Automation System**

A smart payment reminder automation workflow built using n8n, OpenAI, Google Sheets, Gmail, and Telegram.
This system automates payment follow-ups by generating professional AI-based reminder messages, sending automated emails, detecting overdue payments, and escalating alerts through Telegram notifications.

🚀 **Features**

Automated payment reminder workflow

AI-generated professional reminder messages

Google Sheets integration for payment tracking

Automated email notifications using Gmail

Overdue payment escalation alerts via Telegram

Automatic status updates in Google Sheets

No-code/low-code automation using n8n

🛠️ **Tech Stack**

n8n

OpenAI API

Google Sheets API

Gmail API

Telegram Bot API

📌 **Workflow Overview**

Monitor new entries from Google Sheets

Check customer payment status

Identify unpaid payments

Generate personalized reminder messages using AI

Send reminder emails automatically

Detect overdue payments

Send Telegram escalation alerts

Update reminder status in Google Sheets

⚙️** Workflow Components**

Google Sheets Trigger

Monitors newly added customer payment records.

IF Condition

Checks whether the payment status is marked as Unpaid.

AI Agent (OpenAI)

Generates short, professional, and polite payment reminder messages dynamically.

Gmail Node

Sends automated payment reminder emails to customers.

Overdue Detection Logic

Checks whether the due date has passed by more than 7 days.

Telegram Notification

Sends escalation alerts for overdue payments.

Google Sheets Update

Updates the customer status to Reminder Sent after successful email delivery.
